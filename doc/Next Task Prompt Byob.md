## 📝 技術筆記

### SEO優化的技術要點

**圖片SEO優化：**
1. **Alt文字**：描述性文字，包含關鍵字，50-125字
2. **檔案名稱**：包含關鍵字的描述性名稱
3. **標題和說明**：提供額外的SEO價值
4. **技術規格**：檔案大小、格式、尺寸優化

**關鍵字策略：**
1. **Focus Keyword**：每頁面一個主要關鍵字
2. **關鍵字密度**：2-3%，自然分布
3. **長尾關鍵字**：包含相關的長尾關鍵字
4. **標題結構**：H1-H6層級清晰

**Rank Math SEO設定：**
1. **Meta描述**：150-160字，包含關鍵字
2. **標題標籤**：60字以內，包含關鍵字
3. **結構化資料**：Schema標記優化
4. **內部連結**：相關頁面互相連結

**外部連結策略：**
1. **權威網站**：連結到相關的權威網站
2. **相關性**：確保連結內容與頁面相關
3. **錨文字**：使用描述性的錨文字
4. **品質控制**：避免低品質連結
